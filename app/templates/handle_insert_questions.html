Tải lên file Excel đề thi (MCQ)

Lưu ý về cấu trúc file Excel:
File Excel phải có các cột sau (tiêu đề chính xác, viết hoa chữ cái đầu):
- "Nội dung câu hỏi" (bắt buộc)
- "Đáp án A", "Đáp án B", "Đáp án C", "Đáp án D" (bắt buộc)
- "Đáp án đúng" (bắt buộc, ví dụ: A, B, C, D)
- "Mức độ khó" (tùy chọn, kiểu số nguyên, ví dụ: 1, 2, 3...)
- "Chapter" (tùy chọn, kiểu số nguyên, ví dụ: 1, 2, 3...)

Đối với "Mức độ khó" và "Chapter", nếu bạn để trống ô trong Excel, giá trị sẽ được lưu là NULL trong database.
